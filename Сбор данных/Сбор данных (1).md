# Файлы

Самые распространенные форматы файлов для работы csv, xlsx.
Чтобы открыть их в питоне в качестве DataFrame достаточно знать путь к файлу:
``` python
import pandas as pd
df = pd.read_excel('file.xlsx')
df = pd.read_csv('file.csv')
```
