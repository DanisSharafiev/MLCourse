Пороговые значения (Threshold) (сюда может попасть ML-фильтрация)

``` python
import pandas as pd

df = pd.DataFrame({'Product': ['A', 'B', 'C', 'D'],
                   'Rating': [4.5, 3.2, 4.8, 2.9]})

df[df['Rating'] > 4.0]
```
Output:
``` python
Product Rating 0 A 4.5 2 C 4.8
```
