One-hot Encoding

``` python
import pandas as pd

df = pd.DataFrame({'Sword': ['cheap', 'expensive', 'acceptable', 'cheap']})
pd.get_dummies(df, columns=['Sword'])
```

```
   Sword_acceptable Sword_cheap Sword_expensive 
0             False        True           False 
1             False       False            True 
2              True       False           False 
3             False        True           False
```

``` python
from sklearn.preprocessing import LabelEncoder
  
df = pd.DataFrame({'Sword': ['cheap', 'expensive', 'acceptable', 'cheap']})
encoder = LabelEncoder()
encoder.fit_transform(df['Sword'])
```

``` python
df = pd.DataFrame({'Цвет': ['Красный', 'Синий', 'Красный', 'Зеленый'],
                   'Цена': [100, 150, 120, 130]})

mean_target = df.groupby('Цвет')['Цена'].mean()
df['Цвет_код'] = df['Цвет'].map(mean_target)
```
